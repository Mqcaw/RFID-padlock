RFID Padlock API - example commands


Read all students (PowerShell)
[~] $ Invoke-RestMethod -Uri "http://IP_ADDRESS:PORT/api/students" -Method Get
Create new student (PowerShell)
[~] $ Invoke-RestMethod -Uri "http://IP_ADDRESS:PORT/api/students" -Method Post -Headers @{"Content-Type"="application/json"} -Body ('{"id":ID, "name":"NAME", "grade":GRADE}')
Get Student's key card from name (PowerShell)
[~] $ $students = (Invoke-RestMethod -Uri "http://IP_ADDRESS:PORT/api/students" -Method Get); $student = $students | Where-Object { $_.name.Trim().ToLower() -eq "name" } | Select-Object -First 1; $keycards = (Invoke-RestMethod -Uri "http://IP_ADDRESS/api/key_cards" -Method Get); $keycard = $keycards | Where-Object { $_.id -eq $student.keyCardId } | Select-Object -First 1; $keycard
